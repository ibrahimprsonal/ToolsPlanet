# Tools Planet

A collaborative repository for creating awesome web tools using HTML, CSS, and JavaScript.

## Setup Instructions

1. Clone the repository: `git clone <repository-url>`
2. Open `index.html` in your browser.
3. Start coding!

## Git Collaboration Tips

- Create a new branch for each feature: `git checkout -b feature-name`
- Add changes: `git add .`
- Commit changes: `git commit -m "Add feature-name"`
- Push to GitHub: `git push origin feature-name`
- Open a Pull Request (PR) to merge into the `main` branch.
